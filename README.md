# Projeto-Relat-rio-Acad-mico

Projeto onde desenvolvi um script onde consigo manipular dados e gerar resultados em .xlsx para o retorno.

# Sobre o Projeto

Este projeto foi desenvolvido como parte do módulo de manipulação de dados do curso python automático. O objetivo principal era aprender a manipular dados e exportalos como .xlsx.

# Tecnologias Utilizadas
* **Linguagem:** Python 3.10
* **Framework/Lib:** openpyxl

# Como Executar o Projeto

Para rodar este projeto localmente, siga os passos abaixo:

Baixe o arquivo .py e modifique o nome do arquivo "alunos projeto.xlsx" para o seu arquivo de preferência, ou se quiser o arquivo de exemplo, fique a vontade.
